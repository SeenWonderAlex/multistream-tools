What is Multistream Tools?
Multistream Tools, created by SeenWonderAlex, are tools that can help you handle streaming in multiple platforms such as Twitch and YouTube. These are intended to assist you into handling multiple chats, without manually running the other. Not many tools are provided, but more are expected to develop in the near future!
What Tools We Provide?
  • Combined Twitch & YouTube Polls
  • Monitor Twitch Channel Point Redemptions
    • Connecting Accounts

    What Tools We Provide?

    Combined Twitch & YouTube Polls

    Combined Twitch & YouTube Polls is a tool to help combine results from multiple platforms at once. No need to do math anymore. This is for you to simply create a poll and see all the votes at once.

    This tool also offers the ability to save and load Poll presets, in order to run polls quicker and easier. It's a time saver for running and getting feedback, without changing the user's end.

    Requires either a Twitch or a YouTube account that has access to live streaming. This tool does not require you to connect both accounts in order to run polls on one platform.
    NOTICE: When left-clicking on the "Connect" button on Twitch, we'll ask you to authenticate Twitch for our other tool as well. If you are planning to only use this tool, please right click the button instead.
    For YouTube, we'll only ask you to "Manage your YouTube Account" in order to retrieve broadcasts and run polls.

    Monitor Twitch Channel Point Redemptions

    This is a Twitch-Only feature that's offered to monitor if a Channel Point redemption is completed or refunded. Please note that it will only monitor redemptions if you have this page open.

    NOTICE: When left-clicking on the "Connect" button on Twitch, we'll ask you to authenticate Twitch for our other tool as well. If you are planning to only use this tool, please right click the button instead.

    Connecting Accounts

    When connecting a platform by clicking 'Connect', you are always asked to verify what permissions we need to establish the tool's functionality. Once connected, you are able to use the tools we provide for your intention.

    You can always disconnect by clicking 'Disconnect' or going in your account connections and disconnect our service. Read our Privacy Policy on how we handle the data and for more information.
