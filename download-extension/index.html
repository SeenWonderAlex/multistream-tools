How to Install This Extension on Chrome?
If you're lost, try to read this support page for instructions
1. Download the .zip file. Once downloaded, extract the zip file (do not create a separate folder, there's already a folder inside).
2. Navigate to chrome://extensions/
3. Enable Developer Mode
4. Click Load unpacked and select the extension folder (the one you extracted).
5. Close the tab. Refresh the Polling tool. You're good to go!


How does this extension & website works?
Since there's no official API for YouTube Polls, this extension takes the UI elements from the YouTube chat, reads its needed data, and sends it over to the main website.

Therefore, the website combines the total votes between two platforms and calculate the combined votes for each choice.
