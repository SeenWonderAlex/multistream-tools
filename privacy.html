Our Privacy Policy with Twitch API

We use Twitch API to provide our service using your Twitch account authenticated with us. When you authenticate using Twitch's OAuth2 screen, the authentication token, provided by Twitch services, would be used to serve its functionality, but it will be stored in your browser and no where else.

Usage of Twitch's Data

We make use of Twitch's Data for the available services listed in our website. This includes, but not limited to: Multistream Chat, Change Title All At Once, and Combined Polls.

When you authenticate, we'll ask for the following permissions:

  • Send live Stream Chat and Rooms messages.
    This is necessary for utilizing the "Broadcast" feature in Multistream Chat.
  • Perform moderation actions in a channel.
    This is necessary for utilizing the Mod Buttons in Multistream Chat.
  • Manage your channel's broadcast configuration, including updating channel configuration and managing stream markers and stream tags.
    This is necessary for the Change Title All At Once feature.
  • Manage raids on your channel.
    This is currently used to "Undo" or cancel any raids done by an editor.
  • Send announcements in channels where you have the moderator role.
    This is used to allow the user utilize the /announce command from Twitch.
  • Delete chat messages in channels where you have the moderator role.
    This is used as part of the Mod Buttons component in Multistream Chat to delete messages.
  • Grant or remove the moderator role from users in your channel.
    This is currently used to "Undo" or revert a role removal from a separate Lead Moderator or user.
  • Grant or remove the VIP role from users in your channel.
    This is currently used to "Undo" or revert a role removal from a separate Lead Moderator or user.
  • Read the list of emotes you can use.
    This is necessary for utilizing emotes in Multistream Chat.
  • Read unban requests in channels where you have the moderator role.
    This is necessary for mod actions to appear in Multistream Chat, as part of a required permission.
  • Get a list of all subscribers to your channel and check if a user is subscribed to your channel.
    This is used for the usage of sub badges to work correctly in Multistream Chat.
  • View Channel Points rewards and their redemptions on your channel.
    This is used in order for Channel Point Redemptions to appear in Multistream Chat.
  • Manage your channel’s polls.
    This is necessary for the Combined Polls feature.
  • View live Stream Chat and Rooms messages.
    This is necessary for the Multistream Chat feature.
  • View your channel's moderation data including Moderators, Bans, Timeouts and Automod settings.
    This is used to make sure mod actions appear in Multistream Chat.
  • Ban or unban users in channels where you have the moderator role.
    This is used as part of the Mod Buttons component in Multistream Chat to ban & timeout users.
  • Read non-private blocked terms in channels where you have the moderator role.
    This is necessary for mod actions to appear in Multistream Chat, as part of a required permission.
  • Manage chat settings in channels where you have the moderator role.
    This is used to configure chat to follower-only, sub-only, etc.
  • Read the list of VIPs in channels where you have the moderator role.
    This is necessary to allow VIP statuses to show, as well for mod actions to appear in Multistream Chat, as part of a required permission.
  • Read the list of moderators in channels where you have the moderator role.
    This is necessary to allow VIP statuses to show, as well for mod actions to appear in Multistream Chat, as part of a required permission.
  • Manage shoutouts in channels where you have the moderator role.
    This is used to view shoutouts in chat.
  • Read chat messages and appear in chat as your user.
    This is necessary for the Multistream Chat feature.
  • Write chat messages as your user.
    This is necessary for the Multistream Chat feature.
  • Warn users in channels where you have the moderator role.
    This is used as part of the Mod Buttons component in Multistream Chat to warn users.

Usage of YouTube's Data

Multistream make use of YouTube's Data by utilizing its services such as Multistream Chat, Change Title All At Once, and Combined Polls.

You may notice that we will ask you for the following permission:

  • Manage your YouTube account.

We make use of this permission for the following:

  • For Combined Polls, you are able to submit, manage, and delete live chat polls to your livestreams.
  • When you interact with "Load from Recent", Multistream Tools connects to YouTube to grab the recent livestream URL.
  • We gather YouTube Live Chat Events to utilize Multistream Chat, as well as the "Broadcast" feature to send messages.

  • For the combined view count, Multistream Tools connect to YouTube regularly about your stream in order to get the concurrent amount of viewers.
  • When you interact with the Mod Buttons, you are able to ban, timeout, or delete messages from users.

Anything we do is provided by you, and we do not make automatic requests, or falsify the interaction you intend.
